# Лаболаторна робота #3
#### Створено декоратор `memo`. Він дозволяє кешувати результати функції.

<div style="display: flex; gap: 20px;">
<div style="flex: 1;">

**main.py**
```python 
from mem import memo

@memo()
def multiply(x):
    print("calc forr", x)
    return x*x

print(multiply(3))
print(multiply(3))
print(multiply(5))
print(multiply(5))
print(multiply(5))
```
</div>
<div style="flex: 1;">

**output:**
```
calc forr 3
9
9
calc forr 5
25
25
25
```
</div>
</div>
Отримані дані підтверджують роботу кешу: повторні виклики з однаковими значеннями не запускають процес обчислення. Замість цього система звертається до хешу і повертає раніше збережене значення, що значно скорочує час виконання програми.

На даному етапі кеш є необмеженим. Це може призвести до надмірного використання оперативної пам'яті при великій кількості різних вхідних даних.
